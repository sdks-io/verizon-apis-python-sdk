
# Device

Identifies a particular IoT device.

## Structure

`Device`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `id` | `string` | Required | Device identifier. |
| `kind` | `string` | Required | Device kind identifier. |

## Example (as JSON)

```json
{
  "id": "864508030026238",
  "kind": "IMEI"
}
```
